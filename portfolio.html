7 day Weather forecast

The 7-Day Weather Forecast Webpage is a simple yet effective tool for checking the weather forecast for the upcoming week. The webpage provides users with essential weather information for each day, including the date, weather condition (such as sunny, cloudy, rainy, etc.), temperature range, and displays the weather charts also.

Weather Detection Using Machine Learning

Weather Detection Using Machine Learning aimed at accurately identifying weather conditions based on images. The project utilizes machine learning algorithms to analyze images of the sky, landscapes, or weather instruments and classify them into different weather categories such as sunny, cloudy, rainy,foggy or thunderstorm.
